# ハイライト
ドキュメントの更新により、Azure Document Intelligenceや言語サービスの各機能が強化され、詳細な比較情報や新機能が追加されています。特にPII関連のドキュメントやプレイグラウンドの新機能が充実しており、ユーザーはより具体的なユースケースへの適用が可能になります。

## 新機能
- Azure Document IntelligenceおよびContent Understandingの比較表の追加。
- 言語検出およびPII処理のための新しいプレイグラウンドが複数導入。
- テキスト、会話、ドキュメントにおけるPII検出機能の向上。
- Visual Studio Codeとの統合機能の一部サービスへの追加。
  
## 破壊的変更
- 直接的な破壊的変更は確認されていません。

## その他の更新
- 各ドキュメントの日付やサービス名称、APIバージョンの情報更新。
- Azure Document Intelligenceのリリース履歴や機能改善に関するアップデート。
- クイックスタートガイドやAPIコール関連の画像、ファイル名の調整。
- Azure Languageにおける新機能の紹介とプレビュー機能の追加。

# インサイト
最近のAzureサービスに関するドキュメントの大幅なアップデートは、概念ドキュメントやプレイグラウンドの導入などを通じてAzureのさまざまな機能をより具体的に理解できるように設計されています。特に、PII（個人を特定できる情報）に関連する機能に関して、APIとプレイグラウンドの両面での情報が充実しており、ユーザーがプライバシー保護に特化した機能を手軽に試せる環境が構築されています。

Azure Document IntelligenceとContent Understandingの比較表の追加は、ユーザーが異なるサービスを正確に比較し、特定のユースケースに最適なモデルを選択する助けとなります。

新たに追加されたプレイグラウンドは、実際にサービスを試しつつフィードバックを得る場を提供し、特にテキストや会話においてのPII処理をコーディング不要で試せるという、実務に即したアプローチが可能です。また、Visual Studio Codeとの連携機能追加は、開発者がAzureの機能をより迅速に採用できるようにし、開発環境にスムーズに設定を移行することを可能にしました。このような更新により、ドキュメントの視覚的、実用的な質が向上し、Azureのユーザーエクスペリエンスも向上することが期待されます。
